# 課金とサブスクリプション

Lokaはシートベースのサブスクリプションモデルを採用しています。スクール向けのCreatorプランを選び、スクールの成長に合わせてエデュケーターシートとラーナーシートを追加します。

## Loka Creatorプラン

| プラン | 料金 | 内容 |
|------|-------|-------------|
| **Loka Creator DIY** | $6/月 | エデュケーターシート1席 + ラーナーテストシート1席。セルフサービスのセットアップで、すべて自分で設定。 |
| **Loka Creator DFY** | $29/月 | DIYの全内容 + Done-For-Youのプラットフォーム設定 + 月2時間のテクニカルサポート。 |
| **Loka Creator Aspire** | $199/月 | DFYの全内容 + 月2時間のファウンダーコーチングで教育ビジネスを成長。 |

すべてのプランに、Lokaプラットフォームの全機能が含まれます：Living Textbook、AIツール、予約システム、進捗追跡、各種連携。

### 無料トライアル

Lokaが合っているか分からない場合は、[14日間の無料トライアル](/ja/getting-started/free-trial)から始められます。クレジットカード不要で、DIYプランのすべての機能にフルアクセスできます（エデュケーターシート1席、ラーナーテストシート1席付き）。トライアル終了後、有料プランにアップグレードして継続できます。

## エデュケーターシート

スクール内の各エデュケーターには専用のシートが必要です。

- **料金：** エデュケーターシートあたり$6/月
- **課金方法：** 標準の月額Stripeサブスクリプション
- **含まれるシート：** Creatorプランにはエデュケーターシート1席が含まれます
- **追加方法：** 新しいエデュケーターを追加する際に、Stripeチェックアウトを通じて追加シートを購入
- **キャンセル：** 手動 -- 不要になったエデュケーターシートを自分でキャンセル

## ラーナーシート

アクティブなラーナーにもシートが必要ですが、ラーナーの課金は仕組みが異なります。

- **料金：** ラーナーシートあたり$6/月
- **課金方法：** ラーナーがレッスン料を支払うと自動更新
- **自動キャンセル：** ラーナーがレッスン料の支払いを停止すると、シートは更新されません
- **手動管理不要：** ラーナーシートはレッスン料の支払いに連動して有効化・無効化されます

このモデルにより、非アクティブなラーナーに対して料金が発生することはありません。シートはアクティブな生徒数に応じて自然にスケールします。

## 課金ページ

すべての支払いはStripeを通じて処理されます。プロフィールドロップダウン（ヘッダー右上のメニュー）の **Billing** リンクから課金情報にアクセスできます。

課金ページには、現在のサブスクリプション状況とシート情報が表示されます。

## サブスクリプションの管理

### プランの変更

Creatorプランのアップグレードまたはダウングレードについては、Lokaチームにお問い合わせください。次の請求サイクルの開始に合わせてプラン変更を調整できます。

### プランのキャンセル

Loka Creatorサブスクリプションをキャンセルするには：

1. プロフィールドロップダウンから **Billing** に移動
2. **Cancel Subscription** をクリック
3. キャンセルリクエストがLokaチームに送信され、処理されます
4. 現在の請求期間の終了までアクセスが継続されます

:::caution
Creatorプランをキャンセルすると、現在の請求期間の終了後にスクール内のすべてのエデュケーターおよびラーナーアカウントが無効化されます。
:::

### シートの追加・削除

- **エデュケーターシートの追加：** **Users > Manage LP** から新しいエデュケーターを追加 -- プロセス中にシートサブスクリプションが作成されます。
- **エデュケーターシートの削除：** 管理パネルのエデュケーターのプロフィールからシートサブスクリプションをキャンセルします。
- **ラーナーシート：** 自動管理 -- レッスン料の支払い時に有効化され、支払いが停止すると無効化されます。
