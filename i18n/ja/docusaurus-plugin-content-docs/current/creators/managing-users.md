# エデュケーターとラーナーの管理

Loka Creatorとして、スクール内の人材 -- 指導を行うエデュケーターと学習するラーナーの両方 -- を管理します。

## エデュケーターの追加

スクール内の各エデュケーターには **エデュケーターシート** が必要です（シートあたり$6/月）。Loka Creatorプランにはデフォルトでエデュケーターシート1席が含まれています。

新しいエデュケーターを追加するには：

1. 管理サイドバーの **Users > Manage LP** に移動
2. **Add New Teacher** をクリック
3. 名前、メールアドレス、初期設定を入力
4. アカウント設定リンク付きの招待メールがエデュケーターに届きます

有効化後、エデュケーターは以下にアクセスできる専用ダッシュボードを取得します：

- レッスンスケジュールとLiving Textbook
- AI搭載のレッスンツール（ARP生成、アセスメント、フィードバック）
- 担当ラーナーの管理
- Canvasとホワイトボードツール

### エデュケーターの削除

エデュケーターを削除するには：

1. 管理サイドバーの **Users > Manage LP** に移動
2. エデュケーターの名前をクリック
3. シートサブスクリプションのキャンセルまたはアカウントの削除が可能です

シートサブスクリプションをキャンセルすると、月額課金が停止します。エデュケーターを削除すると、ユーザーアカウントが削除されます。過去のセッションのレッスン記録はシステムに保持されます。

## ラーナーアカウントの管理

ラーナーは招待またはサインアップリンクを通じてスクールに参加します。

### ラーナーの招待

1. 管理サイドバーの **Users > Manage Learners** に移動
2. **Add Learner** をクリック
3. ラーナーの詳細情報（名前、メールアドレスなど）を入力
4. アカウント作成の手順が記載されたメールがラーナーに届きます

### ラーナーシート

アクティブなラーナーそれぞれに **ラーナーシート** が必要です（$6/月）。エデュケーターシートとは異なり、ラーナーシートはレッスン料の支払い時に自動更新されます。ラーナーがレッスンの予約を停止すると、シートサブスクリプションは更新されません。

これにより、ラーナーシートを手動でキャンセルする必要はありません -- ラーナーが非アクティブになると自然に解約されます。

### ラーナー詳細の確認

ラーナー管理ページから確認できる項目：

- プロフィール情報と連絡先
- レッスン履歴と今後の予約
- 進捗データと現在のレベル
- シートサブスクリプション状況
- クレジット残高

## ユーザーステータス

ユーザーはシンプルなアクティブ/非アクティブのステータスを持ちます：

| ステータス | 意味 |
|--------|---------|
| **アクティブ** (status = 1) | ログインしてプラットフォームを利用可能 |
| **非アクティブ** (status = 0) | ユーザーアカウントが無効化されている |

プラットフォームの全機能を利用するには、アクティブステータスに加えてアクティブなシートサブスクリプションが必要です。

## オンボーディング

新しいエデュケーターを追加すると、ログイン手順が記載されたウェルカムメールが届きます。その後の流れ：

1. 提供されたログイン情報でログイン
2. プロフィールの完成（名前、タイムゾーン、指導に関する詳細）
3. シートサブスクリプションが有効になると、エデュケーターダッシュボードの利用が可能に

予約ページから登録するラーナーは、登録プロセス中にプロフィールを完成させます。
