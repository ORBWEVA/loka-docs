# Loka Creatorの概要

**Loka Creator** とは、Lokaプラットフォーム上のスクールのオーナーまたは管理者です。Creatorとして、スクールの設定、エデュケーター、ラーナー、課金をすべてコントロールできます。

## Loka Creatorとは

Loka Creatorは、スクールを運営する人と考えてください。自分のビジネスを管理するソロの語学教師かもしれませんし、エデュケーターチームを統括するアカデミーオーナーかもしれません。いずれの場合も、Creatorの役割で、すべてを一か所から管理するツールが提供されます。

Loka Creatorプランに登録すると、以下が得られます：

- Lokaプラットフォーム上に自分専用のスクールインスタンス
- スクールのあらゆる側面を管理する管理パネル
- エデュケーターとラーナーを追加する機能
- 課金、設定、連携の管理権限

## 管理内容

Loka Creatorとして管理する項目：

### エデュケーター
- スクールへのエデュケーターの追加・削除
- 各エデュケーターには、レッスンツール、Living Textbook、AI機能、生徒管理を備えた専用ダッシュボードが付与されます
- エデュケーターシートの管理（各シートは別個のサブスクリプション）

### ラーナー
- ラーナーをスクールに招待
- ラーナーの進捗、レッスン履歴、アカウント状況の確認
- ラーナーシートの課金管理

### 課金＆サブスクリプション
- Loka Creatorプラン（DIY、DFY、Aspire）の選択
- エデュケーターシートの追加購入
- 課金状況の確認と変更依頼

### プラットフォーム設定
- 予約ルール、タイムゾーン、スケジュールの設定
- 連携の設定（Stripe、Zoom、Zoho Invoiceなど）
- ホワイトラベルオプションの設定（スクール名、ロゴ）

## 管理パネルへのアクセス

ログイン後、ナビゲーションメニューに **管理パネル** が表示されます。管理パネルでアクセスできる項目：

- **ユーザー** -- エデュケーター（「Manage LP」配下）とラーナー（「Manage Learners」配下）の管理
- **設定** -- スクール全体の設定、連携、予約ルールの管理
- **課金** -- ヘッダーのプロフィールドロップダウンからアクセス。サブスクリプションの管理
- **プロフィール** -- 管理者プロフィールとパスワードの更新（ヘッダーのドロップダウンからアクセス）

:::tip
自分自身もレッスンを行う場合、管理パネルとエデュケーターダッシュボードは別々のビューで、異なるログイン情報でアクセスします。
:::
