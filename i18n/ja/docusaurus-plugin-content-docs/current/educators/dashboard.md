# ダッシュボードの概要

エデュケーターダッシュボードは、Lokaにおける指揮センターです。レッスンの開始、生徒の管理、収益の追跡、すべての指導ツールへのアクセスがここから行えます。

## ダッシュボードのタブ

ダッシュボードは3つのメインタブで構成されています。

### 収益（マネジメント）

**収益** タブでは、指導のビジネス面を把握できます：

- 選択期間の総収益と日次平均
- トップ5サービスチャート（予約数順）
- トップ5生徒チャート（収益順）
- 日付範囲のフィルタリング（直近30日、60日、90日、またはカスタム範囲）

### Living Textbook（Canvas）

**Living Textbook** タブは、レッスンが形になる場所です。ここから[Living Canvas](/educators/canvas)にアクセスできます。以下のことが可能です：

- 今後のセッションと過去のセッションをクイックアクション（セッション開始 / セッション確認）付きで表示
- 生徒との新しいセッションを開始（アクティブなサブスクリプションが必要）
- 過去のセッションCanvasを確認
- 保存済みのメモ、キーワード、Active Recall Pairs（ARP）にアクセス
- CanvasをPDFまたはMP3でエクスポート

### 検索（キーワード）

**検索** タブでは、すべてのセッションにわたるキーワードや語彙を検索・確認できます。

## 主要指標

収益タブでは、選択期間の指標が表示されます：

| 指標 | 説明 |
|--------|-------------|
| **総収益** | 完了したレッスンからの総売上 |
| **日次平均** | 期間中の平均日次収益 |
| **所要時間** | ログされた指導時間の合計 |
| **トップサービス** | 予約数上位のレッスンタイプ |
| **トップ生徒** | 収益上位の生徒 |

## クイックアクセス機能

- **セッション開始** -- Living TextbookタブからレッスンをLiving Canvasで開始（アクティブな[Loka Creatorサブスクリプション](/educators/earnings)が必要）
- **iCalフィード** -- LokaのスケジュールをiCalフィードとしてエクスポートし、Google Calendarやその他のカレンダーアプリで利用
- **メール通知** -- 予約確認、キャンセル、支払いアラートがメールで送信されます

## はじめに

ダッシュボードを初めて使う場合：

1. プロフィールページで[プロフィール設定](https://thelivingtextbook.lokalingo.com/teachers/profile)を完了する
2. レッスン管理ページから[空き状況を設定](/educators/schedule)する
3. プロフィール > AI連携セクションから[AIプロバイダーキーを追加](/educators/ai-features)する
4. 予約リンクを共有して、ラーナーが登録・レッスン予約できるようにする

:::tip
ダッシュボードのURLをブックマークしておくと、すばやくアクセスできます。スマートフォンのホーム画面に追加すれば、ネイティブアプリのような体験も可能です。
:::
