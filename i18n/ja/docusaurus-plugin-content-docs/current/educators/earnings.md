# 収益と支払い

Lokaは指導による収益を追跡し、収益状況を簡単に把握できるようにします。サマリーの確認、詳細の掘り下げ、支払い方法の設定が可能です。

## 収益の確認

ダッシュボードの **マネジメント** タブに移動して、収益の概要にアクセスします。以下の情報が確認できます：

### 収益サマリー

選択期間の収益のハイレベルビュー：

- **総収益** -- 完了したすべてのレッスンからの総売上
- **日次平均** -- 期間中の平均日次収益
- **所要時間** -- ログされた指導時間の合計

日付範囲ピッカーを使って、今週、今月、カスタム範囲、全期間の収益を確認できます。

### トップサービス

最も人気のあるレッスンタイプを確認：

- サービス名
- サービスごとの予約数
- 予約分布を示す円グラフで表示

### トップ生徒

収益上位の生徒を確認：

- 生徒ごとの総収益
- 収益分布を示す円グラフで表示

## 期間別収益

特定の期間を分析するために収益データをフィルタリングできます：

| ビュー | ユースケース |
|------|----------|
| **今週** | 今週のアクティビティを素早くチェック |
| **今月** | 月次の収益追跡 |
| **カスタム範囲** | 特定期間の比較や税務レポートの作成 |
| **全期間** | Loka利用開始からの全履歴 |

## 支払い方法の設定

Lokaは複数の支払い方法に対応しており、最適な方法で収益を受け取れます。

### Stripe Connect

Lokaは **Stripe Connect** を使った直接支払いに対応しています：

Stripe Connected Accountsを設定して直接支払いを受け取ることができます。通常、スクール管理者が設定します。接続後、レッスン料がStripeアカウントに振り込まれます。

### PayPal

PayPalでの定期送金は、スクール管理者が設定できます。

### Wise（TransferWise）

海外在住のエデュケーター向けに、**Wise** は競争力のある為替レートを提供しています：

プロフィールページの設定からWiseのメールアドレスを登録できます。Wise送金は、海外在住のエデュケーター向けに競争力のある為替レートを提供しています。

:::tip
複数の国のラーナーに指導している場合、国際送金ではWiseが最も良い為替レートと最低手数料を提供するのが一般的です。
:::

## Loka Creatorサブスクリプション

Lokaの指導ツール（Living Canvas、セッション管理、AI機能）を使用するには、アクティブな **Loka Creator** サブスクリプションが必要です：

| プラン | 月額 | 含まれる内容 |
|------|---------------|-----------------|
| **無料トライアル** | 14日間無料 | エデュケーターシート1席 + ラーナーテストシート1席（クレジットカード不要） |
| **Loka Creator DIY** | $6/月 | エデュケーターシート1席 + ラーナーテストシート1席 |
| **Loka Creator DFY** | $29/月 | DIYの全内容 + Done-For-You設定 + 月2時間のテクニカルサポート |
| **Loka Creator Aspire** | $199/月 | DFYの全内容 + 月2時間のファウンダーコーチング |

### 追加シート

- **エデュケーターシート**: シートあたり$6/月 -- スクール内の追加教師向け
- **ラーナーシート**: 生徒あたり$6/月 -- レッスン料の支払いに伴い自動更新

## 課金モデルの仕組み

| シートタイプ | 課金方法 | 自動キャンセル |
|-----------|-----------------|--------------|
| **エデュケーターシート** | 月額Stripeサブスクリプション | なし -- 手動キャンセルが必要 |
| **ラーナーシート** | ラーナーのレッスン料支払い時に更新 | あり -- レッスンが停止すると停止 |

アクティブな生徒数に応じてコストがスケールします。ラーナーが休止すると、シートコストも自動的に一時停止します。

## 税務と報告

Lokaは税務申告に役立つ収益データを提供します：

- 収益タブから日付範囲でフィルタリングした収益履歴を確認
- 30/60/90日のプリセットまたはカスタム日付範囲で税務期間に合わせて確認
- 一括データエクスポートが必要な場合は、スクール管理者にお問い合わせください

:::note
Lokaは源泉徴収や税務アドバイスを提供しません。独立エデュケーターとしての税務義務については、税理士に相談してください。
:::
