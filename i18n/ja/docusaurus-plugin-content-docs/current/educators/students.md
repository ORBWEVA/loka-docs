# 生徒の管理

Lokaでは、スクールでセッションを予約・参加するラーナーの情報を確認できます。

## ラーナーのスクール参加方法

### 予約リンク

ラーナーがスクールに参加する主な方法は、予約ページを通じた方法です：

1. スクールの予約ページURLを見込みのラーナーに共有
2. ページにアクセスし、利用可能なサービスと空き時間を確認
3. Lokaアカウントを作成（またはログイン）
4. 最初のレッスンを予約し、支払いを完了
5. ラーナーが自動的にスクールに追加されます

### 管理パネル

Loka Creatorは、管理パネルの **ユーザー > ラーナー管理** からラーナーを追加できます。ラーナーにはアカウント設定手順を記載したウェルカムメールが届きます。

## ラーナーシート

アクティブな生徒それぞれに **ラーナーシート** サブスクリプションが必要です：

- ラーナーシートは生徒あたり月額$6
- ラーナーがレッスン料を支払うと自動的に更新されます
- ラーナーがレッスンの予約を停止すると、シートサブスクリプションも停止します
- アクティブなシートがない生徒は、新しいレッスンの予約ができません

:::note
ラーナーシートはレッスン料の支払いに連動しています -- 手動で管理する必要はありません。ラーナーが予約・支払いをすればシートはアクティブのまま、予約が止まれば自然に期限切れになります。
:::

## 生徒データの確認

### セッション履歴

ダッシュボードのLiving Textbookタブから、各生徒との過去のセッションを確認できます。各セッションには以下が含まれます：

- Canvasのメモとコンテンツ
- 記録されたキーワードと語彙
- Active Recall Pairs（ARP）
- 添削ペア
- 割り当てられたタスク

### 予約情報

セッションのエントリには以下が表示されます：

- レッスン日時と所要時間
- サービスタイプ
- セッションからの収益
- セッションステータス（予約済み、完了、キャンセル、ノーショー）

## 進捗の確認

生徒のCanvas履歴とARPライブラリを使って進捗を評価できます：

- 添削の傾向を確認 -- 時間とともにエラーが減っているか？
- キーワードを確認 -- どのような語彙を学んできたか？
- セッション頻度を確認 -- 定期的に予約しているか？
