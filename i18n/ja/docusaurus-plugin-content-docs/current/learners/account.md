# アカウント設定

ダッシュボードの **プロフィール** ページから、プロフィールや各種設定を管理できます。

## プロフィール情報

プロフィールには、アカウントで使用される基本情報が含まれます。更新可能な項目：

- **姓名** -- 教師やレッスン記録に表示される名前です。
- **メールアドレス** -- プロフィールに表示されますが、登録後の変更はできません。メールアドレスの変更が必要な場合は、スクールの管理者に連絡してください。
- **電話番号** -- 連絡先の電話番号です（必須）。
- **住所** -- 市区町村、国、郵便番号です。
- **言語** -- プラットフォームの表示言語です。

## タイムゾーン設定

Lokaではすべてのレッスン時間がローカルタイムゾーンで表示されます。正確に設定することが、予約の混乱を防ぐために重要です。

- プロフィールページのドロップダウンからタイムゾーンを設定できます。
- すべての予約確認、リマインダー、カレンダー登録は、選択したタイムゾーンが使用されます。

:::caution
レッスン時間が正しく表示されない場合、最も多い原因はタイムゾーン設定の誤りです。サポートに問い合わせる前に、この設定を確認してください。
:::

## パスワードの変更

パスワードを変更するには、プロフィールページのパスワード欄を更新します：

1. 新しいパスワードを入力します。
2. 新しいパスワードを確認入力します。
3. **保存** または **更新** をクリックします。

パスワードを忘れた場合は、ログインページの **パスワードをお忘れですか？** リンクを使用してください。登録されたメールアドレスにリセットリンクが送信されます。

## 通知

Lokaは、主要なイベントに対してメール通知を自動送信します：

- レッスンの予約またはキャンセル時の予約確認
- 予定されているセッション前のレッスンリマインダー
- リクエスト時のパスワードリセットメール

これらの通知は自動的に送信され、個別にオン・オフを切り替えることはできません。

:::tip
リマインダーの時間が正しく届くよう、タイムゾーンが正確に設定されているか確認してください。
:::
