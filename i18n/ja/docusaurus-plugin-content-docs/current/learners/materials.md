# レッスン教材と録音

レッスンが終わるたびに、Lokaはカバーされた内容の1ページサマリーを提供します。セッションを重ねるごとに成長する、パーソナルなテキストブックと考えてください。

## 1ページレッスンサマリー

完了したレッスンごとに、セッション中の内容をキャプチャしたサマリーページ -- **Canvas** -- が生成されます。含まれる内容：

### Canvasノート
レッスン中の教師によるメモ。主要なディスカッションポイント、説明、セッション中にホワイトボードに書かれた内容がカバーされます。

### キーワード＆語彙
レッスン中に導入された新しい単語やフレーズ。検索可能なタグとして記録され、簡単に復習できます。

### ARP（Active Recall Pairs）
レッスン内容に基づくAI生成の質問と回答のペアです。ARPは、セッションで学んだ重要な語彙、文法、コンセプトの想起力をテストすることで、学習内容の定着を支援します。

### 添削ペア
レッスンからの添削内容で、実際の発言と改善版が並べて表示されます。各添削ペアは、なぜ修正が行われたかを理解できるよう、具体的な文法、語彙、または発音のポイントがハイライトされます。

### タスク
教師が割り当てた宿題やフォローアップタスクです。

## レッスン後の教材へのアクセス

すべてのレッスン教材は、ダッシュボードから利用できます。レッスンは1件ずつ表示され、ナビゲーションボタン（前へ／次へ）でセッション間を移動できます。

- **セッションごとに移動** -- 前へ／次へボタンでレッスン履歴を移動できます。
- **キーワードで検索** -- キーワードセクションを使って、過去のレッスンから特定の語彙を検索できます。

教材は、教師がレッスンサマリーを完成させた後に利用可能になります。

## PDFダウンロード

オフライン学習や印刷のために、レッスンサマリーをPDFとしてダウンロードできます。

1. **マイレッスン** からレッスンサマリーを開く
2. **PDFをダウンロード** ボタンをクリック
3. PDFにはCanvasノート、キーワード、ARP、添削ペア、タスクが整ったフォーマットで含まれます

外出先での復習やスタディパートナーとの共有に便利です。

## MP3音声

Lokaは、テキスト読み上げ機能でレッスン内容のMP3音声ファイルを生成できます。以下に役立ちます：

- **リスニング練習** -- キーワードやフレーズの正しい発音を聞く
- **外出先での復習** -- 通勤中や運動中にレッスンサマリーを聞く
- **記憶の強化** -- 読むだけでなく聞くことで、記憶の定着が向上

MP3を生成するには：

1. レッスンサマリーを開く
2. **MP3** ボタンをクリック
3. 音声ファイルが生成され、ダウンロードまたはストリーミングが可能になります

:::tip
レッスンサマリーを定期的に見返しましょう。間隔反復 -- 時間を空けて繰り返し復習する方法 -- は、知識を長期記憶に定着させる最も効果的な方法のひとつです。
:::
