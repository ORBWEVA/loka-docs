# 進捗の追跡

Lokaは、これまでの成長と次に取り組むべきポイントを可視化します。レッスン、タスク、アセスメントの完了に伴い、進捗は自動的に記録されます。

## レベルシステム

言語レベルはエデュケーターによって設定され、段階的な進級体系に沿っています。レベルは通常、CEFR（A1〜C2）やスクール独自のレベルシステムに準拠しています。

現在のレベルはダッシュボードに表示され、以下を決定します：

- AI生成の復習教材（ARP）の難易度
- 推奨される語彙と文法ポイント
- グループレッスンがある場合のクラス分け

成長に伴い、エデュケーターがレベルを更新できます。

## 構造化カリキュラムレベル

一部のスクールでは、Lokaの構造化カリキュラム機能を使用して、学習を定義されたステージに整理しています。各レベルには説明とアセスメント基準があり、エデュケーターが進捗を評価する際に使用します。準備ができたとエデュケーターが判断した場合、次のレベルに進級できます。

## リワードポイント

一部のスクールでは、レッスン予約時にリワードポイントが付与されます。ポイントは累積され、クレジット残高とともにダッシュボードに表示されます。具体的なリワード体系は、スクールのサービス設定によって異なります。

## クレジット

クレジットは、Lokaでレッスンを予約する際に使用する通貨です。スクールの設定により：

- **クレジット制予約** -- 各レッスンに一定数のクレジットが必要です。クレジットパックを購入または受領し、予約時にクレジットが差し引かれます。
- **サブスクリプション制予約** -- プランに一定数のレッスンが含まれます。プラン上限を超えるレッスンには、クレジットが使用される場合があります。

クレジット残高はダッシュボードで確認できます。

## バッジ

バッジは特定の達成やマイルストーンを表彰します。例：

- **First Steps** -- Lokaで最初のセッションを完了
- **Dedicated Learner** -- 5、10、25、50、100回のセッションを完了
- **On a Roll** -- 3日間連続でセッションを継続
- **Week Warrior** -- 7日間連続でセッションを継続
- **Monthly Master** -- 30日間連続でセッションを継続

獲得したバッジはプロフィールとダッシュボードに表示されます。達成をひと目で確認できる、楽しい機能です。

:::tip
定期的にダッシュボードを確認して、進捗状況を把握しましょう。これまでの成長を見ることは、学習を続ける大きなモチベーションになります。
:::
