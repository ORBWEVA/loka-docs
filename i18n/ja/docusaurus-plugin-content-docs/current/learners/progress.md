# 進捗の追跡

Lokaは、これまでの成長と次に取り組むべきポイントを可視化します。レッスン、タスク、アセスメントの完了に伴い、進捗は自動的に記録されます。

## レベルシステム

言語レベルはエデュケーターによって設定され、段階的な進級体系に沿っています。レベルは通常、CEFR（A1〜C2）やスクール独自のレベルシステムに準拠しています。

現在のレベルはダッシュボードに表示され、以下を決定します：

- AI生成の復習教材（ARP）の難易度
- 推奨される語彙と文法ポイント
- グループレッスンがある場合のクラス分け

成長に伴い、エデュケーターがレベルを更新できます。レベル変更は記録されるため、学習の軌跡を振り返ることができます。

## セッション履歴

ダッシュボードには、完了したすべてのレッスンの記録が保存されています。各セッションで以下を確認できます：

- **Canvasノート** -- レッスン中の教師によるメモ。主要なディスカッションポイントやホワイトボードの内容を含みます。
- **キーワード＆語彙** -- セッション中に導入された新しい単語やフレーズ。定義と例文付き。
- **CIP（Correct & Incorrect Phrases）** -- レッスンからの添削内容。実際の発言と改善版が並べて表示されます。
- **ARP（Active Recall Pairs）** -- レッスン内容の想起力をテストするAI生成の復習ペア。
- **タスク** -- 教師が割り当てた宿題やフォローアップ活動。

セッションコンテンツへのアクセスとダウンロードの詳細は、[レッスン教材と録音](/ja/learners/materials)をご覧ください。

## クレジット

クレジットは、Lokaでレッスンを予約する際に使用されます。レッスンの予約と支払いを継続している限り、ラーナーシートはアクティブな状態が維持されます。レッスン料を支払うと、請求期間中のシートが自動的に更新されます。

ダッシュボードからレッスン履歴と今後の予約を確認できます。

:::tip
レッスン教材を定期的に見返しましょう。Canvasノートの振り返りやARPの練習をセッション間に行うことが、学んだ内容を定着させる最も効果的な方法のひとつです。
:::
