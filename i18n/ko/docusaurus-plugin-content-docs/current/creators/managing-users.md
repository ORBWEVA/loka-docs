# 교육자 및 학습자 관리

Loka Creator로서 학교의 구성원, 즉 수업을 진행하는 교육자와 학습하는 학습자를 관리합니다.

## 교육자 추가

학교의 각 교육자에게는 **교육자 좌석**($6/월)이 필요합니다. Loka Creator 플랜에는 기본으로 교육자 좌석 1개가 포함됩니다.

새 교육자를 추가하려면:

1. 관리 사이드바의 **Users > Manage LP**로 이동합니다
2. **Add New Teacher**를 클릭합니다
3. 이름, 이메일 주소 및 초기 설정을 입력합니다
4. 교육자에게 계정 설정 링크가 포함된 초대 이메일이 발송됩니다

활성화되면 교육자는 다음에 접근할 수 있는 자체 대시보드를 받습니다:

- 수업 스케줄링 및 Living Textbook
- AI 기반 수업 도구 (ARP 생성, 평가, 피드백)
- 배정된 학습자의 학생 관리
- Canvas 및 화이트보드 도구

### 교육자 제거

교육자를 제거하려면:

1. 관리 사이드바의 **Users > Manage LP**로 이동합니다
2. 교육자 이름을 클릭합니다
3. 좌석 구독을 취소하거나 계정을 삭제할 수 있습니다

좌석 구독을 취소하면 월간 결제가 중단됩니다. 교육자를 삭제하면 사용자 계정이 제거됩니다. 과거 세션의 수업 기록은 시스템에 보존됩니다.

## 학습자 계정 관리

학습자는 초대 또는 가입 링크를 통해 학교에 참여합니다.

### 학습자 초대

1. 관리 사이드바의 **Users > Manage Learners**로 이동합니다
2. **Add Learner**를 클릭합니다
3. 학습자의 상세 정보(이름, 이메일 등)를 입력합니다
4. 학습자에게 계정 생성 안내가 포함된 이메일이 발송됩니다

### 학습자 좌석

각 활성 학습자에게는 **학습자 좌석**($6/월)이 필요합니다. 교육자 좌석과 달리, 학습자 좌석은 학습자가 수업료를 결제할 때 자동으로 갱신됩니다. 학습자가 수업 예약을 중단하면 좌석 구독이 갱신되지 않습니다.

따라서 학습자 좌석을 수동으로 취소할 필요가 없습니다 -- 학습자가 비활성화되면 자연스럽게 종료됩니다.

### 학습자 상세 정보 확인

학습자 관리 페이지에서 다음을 확인할 수 있습니다:

- 프로필 정보 및 연락처
- 수업 이력 및 예정 예약
- 진도 데이터 및 현재 레벨
- 좌석 구독 상태
- 크레딧 잔액

## 사용자 상태

사용자는 간단한 활성/비활성 상태를 가집니다:

| 상태 | 의미 |
|------|------|
| **활성** (status = 1) | 로그인하여 플랫폼을 사용할 수 있음 |
| **비활성** (status = 0) | 사용자 계정이 비활성화됨 |

플랫폼의 전체 기능을 사용하려면 활성 상태와 함께 활성 좌석 구독이 필요합니다.

## 온보딩

새 교육자가 추가되면 로그인 안내가 포함된 환영 이메일을 받습니다. 이후 다음을 수행할 수 있습니다:

1. 제공된 자격 증명으로 로그인
2. 프로필 완성(이름, 시간대, 교육 세부 정보)
3. 좌석 구독이 활성화되면 교육자 대시보드 사용 시작

예약 페이지를 통해 등록하는 학습자는 등록 과정 중에 프로필을 완성합니다.
