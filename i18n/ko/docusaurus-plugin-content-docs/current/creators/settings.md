# 플랫폼 설정

Loka Creator로서 비즈니스 요구에 맞게 학교 설정을 구성할 수 있습니다. 모든 설정은 **관리 패널 > 설정**에서 접근합니다.

## 시스템 설정

### 시간대

학교의 기본 시간대를 설정합니다. 이는 다음에 영향을 줍니다:

- 학습자에게 표시되는 가능 시간 슬롯
- 새 사용자 계정의 기본 시간대
- 스케줄링 및 알림 계산

개별 사용자는 자신의 계정 설정에서 별도의 시간대로 변경할 수 있습니다.

### 통화

플랫폼 통화는 현재 USD로 설정되어 있습니다. 통화 설정은 향후 업데이트에서 확장될 수 있습니다.

### 예약 규칙

학습자가 학교에서 수업을 예약하는 방법을 설정합니다:

- **최소 사전 예약** -- 수업이 얼마나 미리 예약되어야 하는지 (예: 24시간)
- **최대 예약 기간** -- 학습자가 얼마나 먼 미래까지 예약할 수 있는지 (예: 2주, 1개월)
- **취소 정책** -- 무료 취소 마감 시간 및 늦은 취소 시 처리 방법
## 연동

Loka는 학교의 기능을 확장하기 위해 여러 타사 서비스와 연결됩니다.

### Stripe

Stripe는 Loka의 모든 결제 처리를 담당합니다. Stripe 연결은 온보딩 중에 설정됩니다.

- **설정 > 연동 > Stripe**에서 연결 상태 확인
- Stripe 대시보드에서 직접 Stripe 계정 관리
- 웹훅 이벤트는 자동으로 처리됩니다 (수업 결제, 좌석 갱신, 취소)

### Zoom

온라인 수업의 미팅 링크를 자동으로 생성하기 위해 Zoom을 연결합니다.

- **설정 > 연동 > Zoom**으로 이동합니다
- Zoom 계정을 승인합니다
- 연결되면 온라인 수업 예약 시 Zoom 미팅이 자동으로 생성되고 확인 이메일에 참여 링크가 포함됩니다

### Zoho Invoice

Zoho Invoice와 연동하여 청구서 발행을 관리합니다. 연결되면 학습자가 학교에 가입할 때 Zoho Invoice에 연락처가 자동으로 생성됩니다.

- **설정 > 연동 > Zoho**로 이동합니다
- Zoho Invoice 계정을 승인합니다
- 새 학습자 등록 시 연락처가 자동으로 생성됩니다

### Asana

Asana 연동은 작업 및 프로젝트 관리에 사용할 수 있습니다. 이 연동은 현재 비활성 상태이며 향후 업데이트에서 활성화될 수 있습니다. 사용 가능해질 때를 위해 설정에 구성 필드가 존재합니다.

## 이메일 설정

Loka는 일반적인 이벤트에 대해 자동 이메일을 발송합니다:

- 환영 / 초대 이메일
- 예약 확인 및 알림
- 수업 요약 알림
- 비밀번호 재설정 이메일

이메일 템플릿에는 학교 이름과 로고가 사전 설정되어 있습니다. 설정에서 알림 이메일 주소를 설정하여 시스템 알림을 받을 수 있습니다.

## 화이트 라벨링

화이트 라벨링 옵션으로 Loka를 자체 플랫폼처럼 만들 수 있습니다. 화이트 라벨링 접근에는 일회성 구매($100)가 필요합니다.

- **학교 이름** -- 플랫폼 전체와 이메일에 표시됩니다
- **로고** -- 학교 로고를 업로드하여 내비게이션 바와 이메일의 기본 Loka 브랜딩을 대체합니다
- **저작권 텍스트** -- 푸터 저작권 텍스트를 커스터마이즈합니다

:::tip
화이트 라벨링은 학습자의 신뢰를 구축하는 데 도움이 됩니다. 플랫폼이 자체 브랜드처럼 보이면 학교의 정체성과 전문성이 강화됩니다.
:::
