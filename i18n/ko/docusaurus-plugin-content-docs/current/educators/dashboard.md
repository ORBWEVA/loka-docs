# 대시보드 개요

교육자 대시보드는 Loka의 총괄 관리 화면입니다. 여기서 수업을 시작하고, 학생을 관리하고, 수익을 추적하며, 모든 교수 도구에 접근할 수 있습니다.

## 대시보드 탭

대시보드는 세 가지 주요 탭으로 구성되어 있습니다:

### 수익 (관리)

**수익** 탭은 교수 활동의 비즈니스 측면을 개요로 제공합니다:

- 선택 기간의 총 수익 및 일일 평균
- 상위 5 서비스 차트 (예약 수 기준)
- 상위 5 학생 차트 (수익 기준)
- 날짜 범위 필터링 (최근 30일, 60일, 90일 또는 사용자 지정 범위)

### Living Textbook (Canvas)

**Living Textbook** 탭은 수업이 펼쳐지는 곳입니다. [Living Canvas](/educators/canvas)의 진입점으로, 여기서 다음을 수행할 수 있습니다:

- 예정된 세션과 지난 세션을 빠른 작업(세션 시작 / 세션 보기)과 함께 확인
- 학생과 새로운 세션 시작 (활성 구독 필요)
- 지난 세션 Canvas 검토
- 저장된 노트, 키워드, Active Recall Pairs (ARP) 접근
- Canvas를 PDF 또는 MP3로 내보내기

### 검색 (키워드)

**검색** 탭에서 모든 세션에 걸쳐 키워드와 어휘를 검색하고 검토할 수 있습니다.

## 주요 지표

수익 탭에서 선택한 기간의 지표를 확인할 수 있습니다:

| 지표 | 설명 |
|------|------|
| **총 수익** | 완료된 수업의 총 매출 |
| **일일 평균** | 기간의 평균 일일 수익 |
| **시간** | 총 교수 시간 |
| **상위 서비스** | 예약 수 기준 가장 인기 있는 수업 유형 |
| **상위 학생** | 수익 기준 가장 가치 높은 학생 |

## 빠른 액세스 기능

- **세션 시작** -- Living Textbook 탭에서 수업을 시작합니다 (활성 [Loka Creator 구독](/educators/earnings) 필요)
- **iCal 피드** -- Loka 스케줄을 iCal 피드로 내보내 Google Calendar 또는 기타 캘린더 앱에서 사용
- **이메일 알림** -- 예약 확인, 취소, 결제 알림이 이메일로 발송됩니다

## 시작하기

대시보드가 처음이라면:

1. 프로필 페이지에서 [프로필 설정](https://thelivingtextbook.lokalingo.com/teachers/profile)을 완료합니다
2. 수업 관리 페이지에서 [가능 시간을 설정](/educators/schedule)합니다
3. 프로필 > AI 연동 섹션에서 [AI 제공업체 키를 추가](/educators/ai-features)합니다
4. 예약 링크를 공유하여 학생이 등록하고 수업을 예약할 수 있도록 합니다

:::tip
대시보드 URL을 북마크하면 빠르게 접근할 수 있습니다. 휴대폰 홈 화면에 추가하면 네이티브 앱과 같은 경험을 얻을 수 있습니다.
:::
