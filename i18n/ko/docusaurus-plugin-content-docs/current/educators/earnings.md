# 수익 및 결제

Loka는 교수 수익을 추적하고 수입을 쉽게 파악할 수 있게 합니다. 요약을 확인하고, 상세 내역을 살펴보며, 지급 방법을 설정할 수 있습니다.

## 수익 확인

대시보드의 **관리** 탭으로 이동하여 수익 개요에 접근합니다. 다음을 확인할 수 있습니다:

### 수익 요약

선택한 기간의 수익 상위 개요:

- **총 수익** -- 완료된 모든 수업의 총 매출
- **일일 평균** -- 기간의 평균 일일 수익
- **시간** -- 기록된 총 교수 시간

날짜 범위 선택기를 사용하여 이번 주, 이번 달, 사용자 지정 범위 또는 전체 기간의 수익을 확인할 수 있습니다.

### 상위 서비스

가장 인기 있는 수업 유형 확인:

- 서비스 이름
- 서비스별 예약 수
- 예약 분포를 보여주는 원형 차트로 표시

### 상위 학생

다음 기준으로 가치가 높은 학생 확인:

- 학생별 총 발생 매출
- 수익 분포를 보여주는 원형 차트로 표시

## 기간별 수익

특정 기간을 분석하기 위해 수익 데이터를 필터링합니다:

| 보기 | 용도 |
|------|------|
| **이번 주** | 이번 주 활동 빠른 확인 |
| **이번 달** | 월간 수익 추적 |
| **사용자 지정 범위** | 특정 기간 비교 또는 세금 보고 준비 |
| **전체 기간** | Loka 가입 이후 전체 이력 |

## 지급 설정

Loka는 가장 편리한 방식으로 수익을 받을 수 있도록 여러 지급 방법을 지원합니다.

### Stripe Connect

Loka는 직접 지급을 위해 **Stripe Connect**를 사용합니다:

직접 지급을 위해 Stripe Connected Account를 설정할 수 있습니다. 일반적으로 학교 관리자가 설정합니다. 연결되면 수업 결제가 Stripe 계정으로 이체됩니다.

### PayPal

PayPal을 통한 정기 송금은 학교 관리자가 설정할 수 있습니다.

### Wise (TransferWise)

해외 교육자의 경우 **Wise**가 경쟁력 있는 환율을 제공합니다:

프로필 페이지 설정에서 Wise 이메일을 등록할 수 있습니다. Wise 송금은 해외 교육자에게 경쟁력 있는 환율을 제공합니다.

:::tip
여러 나라의 학생을 가르치는 경우, Wise가 일반적으로 국제 송금에 가장 좋은 환율과 최저 수수료를 제공합니다.
:::

Loka의 교수 도구(Living Canvas, 세션 관리, AI 기능)를 사용하려면 활성 **Loka Creator** 구독이 필요합니다:

| 플랜 | 월 가격 | 포함 내용 |

| **무료 체험** | 14일 무료 | 교육자 좌석 1개 + 학습자 테스트 좌석 1개 (신용카드 불필요) |
| **Loka Creator DIY** | $6/월 | 교육자 좌석 1개 + 학습자 테스트 좌석 1개 |
| **Loka Creator DFY** | $29/월 | DIY 전체 + Done-For-You 설정 + 월 2시간 기술 지원 |
| **Loka Creator Aspire** | $199/월 | DFY 전체 + 월 2시간 파운더 코칭 |

### 추가 좌석

- **교육자 좌석**: 좌석당 $6/월 -- 학교의 추가 교사용
- **학습자 좌석**: 학생당 $6/월 -- 수업 결제 시 자동 갱신

## 결제 모델 이해

| 좌석 유형 | 결제 방식 | 자동 취소 여부 |
|----------|----------|-------------|
| **교육자 좌석** | 월간 Stripe 구독 | 아니오 -- 수동 취소 필요 |
| **학습자 좌석** | 학습자가 수업료를 결제할 때 갱신 | 예 -- 수업 중단 시 자동 중지 |

이는 활성 학생 기반에 따라 비용이 조정됨을 의미합니다. 학생이 휴식을 취하면 해당 좌석 비용도 자동으로 일시 중지됩니다.

## 세금 및 보고

Loka는 세금 보고에 도움이 되는 수익 데이터를 제공합니다:

- 수익 탭에서 날짜 범위로 필터링된 수익 이력 확인
- 30/60/90일 프리셋 또는 사용자 지정 날짜 범위로 세금 기간에 맞추기
- 대량 데이터 내보내기가 필요한 경우 학교 관리자에게 문의하세요

:::note
Loka는 세금을 원천징수하거나 세금 관련 조언을 제공하지 않습니다. 독립 교육자로서의 납세 의무에 대해서는 세무 전문가와 상담하세요.
:::
