# 스케줄 설정

Loka의 스케줄링 시스템은 학생이 설정된 가능 시간에 따라 수업을 예약할 수 있도록 합니다. 주간 시간을 설정하고, 제약 조건을 정의하며, 개인 캘린더와 모든 것을 연동하세요.

## 주간 가능 시간 설정

정기 스케줄을 구성하려면:

1. **수업 관리** 페이지(교육자 내비게이션에서 접근 가능)로 이동하여 **스케줄 관리** 탭을 선택합니다
2. 수업 가능한 요일을 선택합니다
3. 각 요일에 하나 이상의 시간 슬롯을 추가합니다 (예: 오전 9:00 -- 오후 12:00, 오후 2:00 -- 오후 6:00)
4. **저장**을 클릭하여 가능 시간을 공개합니다

학생은 학교의 예약 페이지에서 열린 슬롯만 볼 수 있습니다.

:::tip
스케줄을 설정하기 전에 **프로필** 페이지 설정에서 시간대를 설정하세요. 모든 시간 슬롯은 교육자의 현지 시간대로 표시되고, 학생에게는 해당 학생의 시간대로 표시됩니다.
:::

## 시간 슬롯

하루에 여러 시간 블록을 설정할 수 있습니다. 점심 시간이나 준비 시간을 확보할 때 유용합니다:

| 요일 | 오전 | 오후 |
|------|------|------|
| 월요일 | 9:00 -- 12:00 | 14:00 -- 18:00 |
| 화요일 | 10:00 -- 12:00 | 13:00 -- 17:00 |
| 수요일 | -- | 14:00 -- 20:00 |

이미 예약된 슬롯은 자동으로 새 학생에게 비가용으로 표시됩니다.

## 스케줄 예외

특정 날짜에 쉬거나 시간을 조정해야 하는 경우 **스케줄 예외**를 사용하여 주간 패턴을 재정의할 수 있습니다:

- **날짜 차단** -- 특정 날을 비가용으로 표시합니다 (휴일, 개인 일정)
- **시간 수정** -- 정기 스케줄에 영향을 주지 않고 특정 날짜의 가능 시간을 변경합니다

예외는 주간 가능 시간보다 우선 적용됩니다.

## 예약 제약 조건

학생이 예약하는 방법과 시기를 세부 조정할 수 있습니다:

### 사전 알림

수업 예약 전 필요한 최소 리드 타임을 설정합니다. 예를 들어, 24시간 사전 알림을 요구하면 준비할 수 없는 막판 예약을 방지할 수 있습니다.

### 취소 정책

학생이 요금 부과 없이 취소해야 하는 사전 시간(시간 단위)을 설정합니다. 예를 들어, 24로 설정하면 학생은 수업 시작 최소 24시간 전에 취소해야 합니다. 이 설정은 예약 시 학생에게 표시됩니다.

## 캘린더 연동

Loka는 기존 캘린더 앱에서 스케줄을 확인할 수 있는 도구를 제공합니다.

### iCal 피드

Loka 스케줄을 모든 캘린더 앱으로 내보냅니다:

1. **수업 관리** 페이지로 이동하여 **캘린더** 탭을 선택합니다
2. **iCal 피드 URL**을 복사합니다
3. 캘린더 앱(Apple Calendar, Google Calendar, Outlook 등)에서 구독 캘린더로 추가합니다
4. Loka 스케줄이 읽기 전용 이벤트로 표시됩니다

### Google Calendar 링크

캘린더 탭에는 Google Calendar를 여는 빠른 링크도 제공되어, Loka 스케줄과 개인 캘린더를 쉽게 교차 확인할 수 있습니다.

:::note
iCal 피드는 단방향입니다 -- Loka 스케줄을 캘린더 앱으로 내보냅니다. Loka는 외부 캘린더의 이벤트를 읽지 않습니다.
:::

## 모범 사례

- **일관성 유지** -- 학생은 매주 예측 가능한 가능 시간을 선호합니다
- **현실적인 시간 설정** -- 준비 시간, 휴식, 행정 업무를 고려하세요
- **예외를 미리 설정** -- 예정된 휴일은 알게 되는 즉시 차단하세요
- **매주 검토** -- 매주 일요일에 스케줄을 확인하여 다가오는 주가 올바른지 확인하세요
