# 학생 관리

Loka는 학교에서 세션을 예약하고 참석하는 학습자의 정보를 확인할 수 있게 해줍니다.

## 학습자가 학교에 참여하는 방법

### 예약 링크

학습자가 참여하는 주요 방법은 학교의 예약 페이지를 통한 것입니다:

1. 예비 학생에게 예약 페이지 URL을 공유합니다
2. 페이지를 방문하여 이용 가능한 서비스와 시간 슬롯을 탐색합니다
3. Loka 계정을 생성합니다 (또는 로그인)
4. 첫 수업을 예약하고 결제를 완료합니다
5. 학습자가 자동으로 학교에 추가됩니다

### 관리 패널

Loka Creator는 관리 패널의 **사용자 > 학습자 관리**에서 학습자를 추가할 수 있습니다. 학습자에게 계정 설정 안내가 포함된 환영 이메일이 발송됩니다.

## 학습자 좌석

각 활성 학생에게는 **학습자 좌석** 구독이 필요합니다:

- 학습자 좌석은 학생당 $6/월입니다
- 학습자가 수업료를 결제하면 좌석이 자동으로 갱신됩니다
- 학생이 수업 예약을 중단하면 좌석 구독도 중단됩니다
- 활성 좌석이 없는 학생은 새 수업을 예약할 수 없습니다

:::note
학습자 좌석은 수업 결제에 연동됩니다 -- 수동으로 관리할 필요가 없습니다. 학생이 예약하고 결제하면 좌석이 활성 상태를 유지합니다. 중단하면 자연스럽게 만료됩니다.
:::

## 학생 데이터 확인

### 세션 이력

대시보드의 Living Textbook 탭에서 각 학생과의 지난 세션을 검토할 수 있습니다. 각 세션에는 다음이 포함됩니다:

- Canvas 노트 및 콘텐츠
- 기록된 키워드와 어휘
- Active Recall Pairs (ARP)
- 정확/부정확 구문 (CIP)
- 배정된 과제

### 예약 정보

세션 항목에는 다음이 표시됩니다:

- 수업 날짜, 시간 및 시간
- 서비스 유형
- 세션 수익
- 세션 상태 (예약됨, 완료, 취소, 노쇼)

## 진도 검토

학생의 Canvas 이력과 ARP 라이브러리를 사용하여 진도를 평가합니다:

- CIP 추이를 확인합니다 -- 시간이 지남에 따라 실수가 줄어들고 있나요?
- 키워드를 검토합니다 -- 어떤 어휘를 학습해 왔나요?
- 세션 빈도를 확인합니다 -- 정기적으로 예약하고 있나요?
