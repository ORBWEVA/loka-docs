# 계정 설정

대시보드의 **프로필** 페이지에서 프로필과 환경 설정을 관리할 수 있습니다.

## 프로필 정보

프로필에는 계정에서 사용되는 기본 정보가 포함됩니다. 다음을 업데이트할 수 있습니다:

- **성명** -- 교사와 수업 기록에 표시되는 이름입니다.
- **이메일 주소** -- 프로필에 표시되지만 등록 후 변경할 수 없습니다. 이메일 변경이 필요한 경우 학교 관리자에게 문의하세요.
- **전화번호** -- 연락처 전화번호입니다(필수).
- **주소** -- 도시, 국가, 우편번호입니다.
- **언어** -- 플랫폼 표시 언어입니다.

## 시간대 설정

Loka는 모든 수업 시간을 현지 시간대로 표시합니다. 올바르게 설정하는 것이 예약 혼동을 방지하는 데 중요합니다.

- 프로필 페이지의 드롭다운에서 시간대를 설정할 수 있습니다.
- 모든 예약 확인, 알림, 캘린더 항목은 선택한 시간대를 사용합니다.

:::caution
수업 시간이 잘못 표시되는 경우, 가장 흔한 원인은 잘못된 시간대 설정입니다. 고객지원에 문의하기 전에 이 설정을 먼저 확인하세요.
:::

## 비밀번호 변경

비밀번호를 변경하려면 프로필 페이지에서 비밀번호 항목을 업데이트합니다:

1. 새 비밀번호를 입력합니다.
2. 새 비밀번호를 확인 입력합니다.
3. **저장** 또는 **업데이트**를 클릭합니다.

비밀번호를 잊은 경우 로그인 페이지의 **비밀번호 찾기** 링크를 사용하세요. 등록된 이메일 주소로 재설정 링크가 발송됩니다.

## 알림

Loka는 주요 이벤트에 대해 자동으로 이메일 알림을 보냅니다:

- 수업을 예약하거나 취소할 때 예약 확인
- 예정된 세션 전 수업 알림
- 요청 시 비밀번호 재설정 이메일

이러한 알림은 자동으로 발송되며 개별적으로 켜거나 끌 수 없습니다.

:::tip
알림 시간이 현지 일정에 맞게 표시되도록 시간대가 올바르게 설정되어 있는지 확인하세요.
:::
